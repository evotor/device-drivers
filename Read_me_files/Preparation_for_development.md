## 1.1.1. Подключение библиотеки для работы с оборудованием к своему проекту.
Для этого в `build.gradle` проекта добавьте ссылку на репозиторий jitpack:

```
allprojects {
    repositories {
        jcenter()
        maven { url 'https://jitpack.io' }
    }
}
```

в модуле `build.gradle` добавьте зависимость и укажите точную версию:

```
dependencies {
    compile 'com.github.evotor:device-drivers:v2.0.1+'
}
```
